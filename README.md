<h2>📘 Descripción</h2>

<p>
  <strong>CV React</strong> es una aplicación web desarrollada con <strong>React</strong> que genera
  una página personal de currículum vitae, pensada como alternativa moderna al CV tradicional en PDF.
</p>

<p>
  El proyecto sirve tanto como:
</p>

<ul>
  <li>Web personal profesional</li>
  <li>Portafolio online</li>
  <li>Ejemplo didáctico de aplicación React bien estructurada</li>
</ul>

<p>
  Está diseñada para ser <strong>clara, responsive y fácilmente ampliable</strong>.
</p>

<h2>🚀 Instalación</h2>

<p>Clona el repositorio e instala las dependencias:</p>

<pre><code>git clone https://github.com/fpalacioschaves/cv-react.git
cd cv-react
npm install
</code></pre>

<hr/>

<h2>▶️ Uso</h2>

<p>Arranca el servidor de desarrollo:</p>

<pre><code>npm run dev
</code></pre>

<p>
  Accede desde el navegador a la URL indicada por Vite (normalmente <code>http://localhost:5173</code>).
</p>

<p>
  El contenido del CV se gestiona directamente desde los componentes y/o ficheros de datos,
  lo que permite personalizarlo fácilmente.
</p>

<hr/>

<h2>🏗️ Build para producción</h2>

<p>Genera la versión optimizada:</p>

<pre><code>npm run build
</code></pre>

<p>
  El resultado se genera en la carpeta <code>dist/</code>, lista para desplegar en cualquier
  hosting estático (GitHub Pages, Netlify, Firebase, etc.).
</p>
